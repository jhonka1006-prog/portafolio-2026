# Jhonkarly Website

Sitio web construido con Vite + React + TypeScript.

## Requisitos

- Node.js 18+
- npm

## Desarrollo local

```sh
npm install
npm run dev
```

## Build de producción

```sh
npm run build
npm run preview
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Dominio

Este proyecto está preparado para desplegarse en:

- https://jhonkarly.com
